# CT-RAM Upload Page

Hosted on GitHub Pages for the CT-RAM Clinical Trial Intake Chatbot.

## Setup

1. Create a GitHub repository named `ct-ram`
2. Upload these files to the repository
3. Go to Settings → Pages → Source: main branch → Save
4. Your upload page will be at: `https://<your-username>.github.io/ct-ram/upload.html`

## Update the prompt

Replace `your git hub name` in the prompt with your actual GitHub username.
